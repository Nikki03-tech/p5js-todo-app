# p5.js Todo App ✨

**GSoC 2026 Demo** - Creative coding meets productivity using p5.js Web Editor tech stack.

## 🌟 Features
- Interactive canvas-based todo list
- p5.js animations & drawing
- Add/delete tasks with mouse clicks
- Visual feedback for GSoC E2E testing demo

## 🛠️ Tech Stack
``` 
p5.js - JavaScript - HTML5 Canvas - CSS Animations
```


## 🎮 Live Demo
1. Save `index.html`
2. Open in p5.js Web Editor: [editor.p5js.org](https://editor.p5js.org)
3. Click canvas to add todos!
